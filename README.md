## Getting Started

Please remove test/graphq.test.ts when modify the code, the test/graphql.test.ts is to test initial server setup works.

### Setup

``` bash
yarn install
```

### Runing in dev server

``` bash
yarn start
```

### Running test

``` bash
yarn test
```
